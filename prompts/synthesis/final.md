# Final Synthesis

Synthesize current news, global brain, swept memory, retrieved raw cases,
counterexamples, cutoff-verified web research, candidate research, and red-team
objections.

For BLIND, use only `web_research.sources` whose timestamps were verified at or
before cutoff. Never use excluded, unverified, or cutoff-after web sources. Price
repository data and D-day outcomes are not available evidence. Treat the
`d_minus_one_market_data` payload as a protocol receipt showing price access was
disabled, not as market data.
