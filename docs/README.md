# NSLAB Docs Map

This directory mixes production prompts, automation prompts, gold bundles, and archived reference material. Automation should use the files below as the current source of truth.

- current production prompt = `docs/research_prompt.md`
- current automation runner prompt = `docs/session_prompt.md`
- current gold output shape = `docs/example2.md`
- do not use `example.md` as gold
- do not use archived prompts or archived examples for production

Archived examples belong under `docs/old/` or `docs/archive/`. They are retained only for comparison and migration history.
